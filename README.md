# client

## Refresh requirements.txt
```
sudo pip3 install pipreqs

# Run in current directory
python3 -m  pipreqs.pipreqs .
```

# Create venv
```
python3 -m venv .venv
source .venv/bin/activate
pip3 install requirements.txt
```
